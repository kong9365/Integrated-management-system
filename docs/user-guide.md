## 시스템 개요

품질관리팀 통합 관리 시스템은 시험장비 예약, 외부인 출입관리, 장비 모니터링 등 품질관리팀의 업무를 통합 관리하는 웹 기반 시스템입니다.

### 주요 특징
- 실시간 데이터 자동 갱신 (5초 간격)
- 직관적인 사용자 인터페이스
- 모바일 및 데스크톱 환경 지원
- CDS 시스템과의 실시간 연동

## 메인페이지 (대기화면)

메인페이지는 시스템의 첫 화면으로, 공지사항과 미디어 콘텐츠를 표시하는 대기화면입니다.

### 주요 기능

#### 1. 공지사항 영역
- **기능**: 공지사항, 생일 축하 메시지, 환영 메시지 표시
- **표시 우선순위**:
  1. 환영 메시지 (isWelcome): 가장 상단에 배너 형태로 표시
  2. 생일 축하 메시지: 환영 메시지 다음에 배너 또는 카드 형태로 표시
  3. 중요 공지사항: 노란색 배경과 빨간색 테두리로 강조 표시
  4. 일반 공지사항: 기본 스타일로 표시
- **공지사항 종류**:
  - **일반 공지사항**: 기본 스타일로 표시
  - **중요 공지사항**: 노란색 배경, 빨간색 테두리, 메가폰 아이콘으로 강조
  - **환영 메시지**: 화려한 그라데이션 배경과 특별한 스타일로 표시
  - **생일 축하 메시지**: 생일 케이크 아이콘과 특별한 스타일로 표시
- **게시 기간 설정**:
  - 영구 게시: 종료일 없이 계속 표시
  - 기간 게시: 시작일과 종료일을 설정하여 특정 기간만 표시

#### 2. 미디어 영역
미디어 영역은 3개의 탭으로 구성되어 있습니다:

##### 광고 탭
- **기능**: YouTube 영상 재생
- **표시 내용**:
  - YouTube 채널의 업로드 영상 목록 자동 재생
  - 설정에서 채널명과 URL 변경 가능
  - 자동재생 옵션 설정 가능
- **사용 방법**:
  - 영상이 자동으로 재생됩니다 (설정에 따라)
  - 하단의 YouTube 채널 링크를 클릭하여 채널로 이동 가능

##### 홍보 탭
- **기능**: 홍보 이미지 슬라이더
- **표시 내용**:
  - 업로드된 홍보 이미지를 슬라이더 형태로 표시
  - 여러 이미지가 있을 경우 좌우 화살표로 이동
  - 하단 인디케이터로 현재 페이지 확인
- **사용 방법**:
  - 좌우 화살표 버튼으로 이미지 이동
  - 터치 스와이프로 이미지 이동 (모바일)
  - 하단 점(인디케이터)으로 현재 페이지 확인

##### 시험실 탭
- **기능**: 시험실 이미지 또는 PDF 문서 표시
- **파일 타입**:
  - **이미지**: PNG, JPG, GIF, WEBP 형식의 이미지 파일
  - **PDF**: PDF 문서 파일
- **표시 방식**:
  - 이미지: 슬라이더 형태로 여러 이미지 표시
  - PDF: iframe으로 PDF 문서 표시
- **사용 방법**:
  - 좌우 화살표 버튼으로 파일 이동
  - 터치 스와이프로 파일 이동 (모바일)
  - 하단 인디케이터로 현재 페이지 확인

#### 3. 실시간 시계
- **기능**: 현재 시간을 실시간으로 표시
- **위치**: 상단 헤더 우측
- **형식**: 시:분:초 (24시간 형식)

#### 4. 시작하기 버튼
- **기능**: 관리 메뉴 페이지로 이동
- **위치**: 화면 하단 중앙
- **사용 방법**: "시작하기" 버튼을 클릭하여 관리 메뉴로 이동

### 설정 기능 (관리자 전용)

상단 우측의 설정 버튼을 클릭하면 다음 기능을 관리할 수 있습니다:

#### 1. 광고 영상 설정
- **YouTube URL**: YouTube 임베드 URL 또는 채널 설정
- **채널명**: YouTube 채널명 입력
- **자동재생**: 페이지 로드 시 자동 재생 여부 설정
- **사용 방법**:
  - YouTube URL 또는 채널명 입력
  - 자동재생 스위치로 재생 옵션 설정
  - 입력 후 자동 저장

#### 2. 홍보 이미지 관리
- **이미지 업로드**: PNG, JPG, GIF, WEBP 형식의 이미지 파일 업로드 (최대 10MB)
- **이미지 삭제**: 업로드된 이미지 삭제
- **사용 방법**:
  1. "이미지 파일 선택" 버튼으로 파일 선택
  2. "업로드" 버튼 클릭
  3. 업로드된 이미지는 목록에 표시됨
  4. 삭제 버튼으로 이미지 삭제 가능

#### 3. 생일자 관리
- **기능**: 생일자 정보 등록 및 관리
- **입력 항목**:
  - 이름 (필수)
  - 생일 월/일 (필수)
  - 부서 (선택사항)
- **자동 기능**:
  - 등록된 생일자의 생일이 되면 자동으로 생일 축하 공지사항 생성
  - 생일 축하 메시지는 특별한 스타일로 표시
- **사용 방법**:
  1. 이름, 생일 월/일 입력
  2. 부서명 입력 (선택사항)
  3. "등록" 버튼 클릭
  4. 수정/삭제 버튼으로 관리

#### 4. 공지사항 관리
- **기능**: 공지사항 등록, 수정, 삭제
- **입력 항목**:
  - 공지사항 내용 (필수)
  - 중요 공지 여부: 노란색 배경과 빨간색 테두리로 강조
  - 환영 메시지 여부: 화려한 스타일로 표시
  - 게시 기간:
    - 영구 게시: 종료일 없이 계속 표시
    - 기간 게시: 시작일과 종료일 설정
- **사용 방법**:
  1. 공지사항 내용 입력
  2. 중요 공지 또는 환영 메시지 여부 선택
  3. 게시 기간 설정 (영구 또는 기간)
  4. "등록" 버튼 클릭
  5. 수정/삭제 버튼으로 관리

#### 5. 시험실 관리
- **기능**: 시험실 이미지 또는 PDF 파일 관리
- **파일 타입 선택**:
  - 이미지: PNG, JPG, GIF, WEBP 형식 (최대 50MB)
  - PDF: PDF 문서 파일 (최대 50MB)
- **주의사항**: 파일 타입 변경 시 기존 파일이 삭제됩니다
- **사용 방법**:
  1. 파일 타입 선택 (이미지 또는 PDF)
  2. 파일 선택 후 업로드
  3. 업로드된 파일은 목록에 표시됨
  4. 삭제 버튼으로 파일 삭제 가능

### 화면 구성

메인페이지는 2열 그리드 레이아웃으로 구성됩니다:
- **좌측**: 공지사항 영역
- **우측**: 미디어 영역 (광고/홍보/시험실 탭)

### 터치 제스처 지원 (모바일)
- **스와이프**: 홍보 이미지와 시험실 이미지를 좌우로 스와이프하여 이동 가능
- **최소 스와이프 거리**: 50px 이상

### 사용 흐름
1. 시스템 접속 시 메인페이지(대기화면) 표시
2. 공지사항과 미디어 콘텐츠 확인
3. 하단의 "시작하기" 버튼 클릭
4. 관리 메뉴 페이지로 이동하여 원하는 기능 선택

## 관리 메뉴

관리 메뉴는 각 페이지 우측상단에 위치하며, 해당 페이지의 기능에 접근할 수 있는 메인 페이지입니다.

### 기능 설명
- **실시간 현황 표시**: 각 기능 카드에 현재 상태가 실시간으로 표시됩니다
  - 시험장비 예약: 금일 예약 건수
  - 외부인 출입관리: 방문 현황 (등록/예약 대기 인원)
  - 시험장비 모니터링: 평균 가동률
  - 시험장비 가동현황: 가동 중인 장비 수

### 사용 방법
1. 원하는 기능 카드를 클릭하여 해당 페이지로 이동합니다
2. 각 카드에는 현재 상태 정보가 표시되어 빠르게 현황을 파악할 수 있습니다
3. 상단의 "대기화면으로" 버튼을 클릭하면 대기화면으로 돌아갑니다

## 시험장비 예약

시험장비 예약 기능을 통해 장비 사용 일정을 관리하고 예약할 수 있습니다.

### 주요 기능

#### 1. 캘린더 탭
- **기능**: 월별 캘린더 형태로 예약 현황을 한눈에 확인
- **사용 방법**:
  - 좌우 화살표 버튼으로 월 이동
  - 위치 필터와 장비 필터로 원하는 예약만 표시
  - 예약된 일정을 드래그 앤 드롭으로 날짜 변경 가능 (시작일만 이동 가능)
  - 기간 예약의 경우 시작일을 이동하면 종료일도 함께 이동하여 기간 길이 유지

#### 2. 예약 신청 탭
- **기능**: 새로운 장비 예약 신청
- **필수 입력 항목**:
  - 위치 선택: 장비가 위치한 장소 선택
  - 장비 선택: 예약할 장비 선택 (위치 선택 후 활성화)
  - 사용자명: 예약자 이름 (로그인 시 자동 입력)
  - 사용 일자: 예약 날짜
  - 사용 목적: 드롭다운 선택 또는 직접입력
    - 정기 시험
    - 안정성 시험
    - 밸리데이션
    - 연구개발
    - 기타
    - 직접입력 (드롭다운에서 선택 시 Input 필드 활성화)
- **선택 입력 항목**:
  - 종료 일자: 기간 예약 시 종료 날짜
  - 시작 시간 / 종료 시간: 사용 시간대
  - 시험 항목: 시험 내용
  - 샘플 수: 시험 샘플 개수
  - 비고: 추가 정보

#### 3. 내 예약 탭
- **기능**: 본인이 예약한 장비 목록 확인 및 관리
- **사용 방법**:
  - 예약 목록에서 본인의 예약 확인
  - 상태가 "대기중"인 예약은 취소 가능
  - 예약 정보 확인 (장비명, 예약일시, 사용시간, 사용목적, 상태)

#### 4. 전체 예약 탭
- **기능**: 모든 사용자의 예약 현황 조회
- **표시 정보**:
  - 장비명
  - 사용자
  - 예약일시
  - 사용시간
  - 사용목적
  - 상태

#### 5. 통계 탭
- **기능**: 장비별 예약 횟수 통계
- **표시 내용**: Top 10 장비의 예약 횟수를 막대 그래프로 표시

### 로그인 기능
- **사전 등록 필요**: 예약 신청 전에 설정에서 사용자 등록 필요
- **로그인 방법**:
  1. 상단 "설정" 버튼 클릭
  2. "사용자 등록" 탭에서 사번과 이름 등록 (관리자 권한 필요)
  3. 메인 화면에서 사번과 이름으로 로그인
- **로그인 후**: 사용자명이 자동으로 입력되어 예약 신청이 편리해집니다

### 설정 기능 (관리자 전용)
- **장비 등록**: 장비 코드, 기계명, 위치 등록 및 관리
- **사용자 등록**: 사번, 이름 등록 및 관리자 권한 부여
- **권한 관리**: 관리자 권한이 있는 사용자만 설정 탭 접근 가능

## 외부인 출입관리

외부인 출입관리 기능을 통해 방문자의 사전예약, 등록 및 출입 현황을 관리합니다.

### 주요 기능

#### 1. 대시보드 탭
- **기능**: 방문등록과 사전방문등록 현황을 통합하여 확인
- **표시 내용**:
  - 방문등록: 오늘 이후 예정된 방문 등록 목록
  - 사전방문등록: 오늘 이후 예정된 방문 예약 목록
- **필터 기능**:
  - 날짜 범위 필터
  - 방문자명, 담당자, 소속, 방문목적 검색
  - 상태별 필터 (대기, 승인, 거부, 완료)

#### 2. 방문 등록 탭
- **기능**: 방문 당일 방문자 등록 및 출입 처리
- **필수 입력 항목**:
  - 방문자명
  - 소속: 드롭다운 선택 또는 직접입력
    - 설정에서 등록된 소속 목록에서 선택
    - "직접입력" 선택 시 Input 필드 활성화
  - 연락처: "-" 없이 숫자만 입력 (예: 01012345678)
  - 방문 목적: 드롭다운 선택 또는 직접입력
    - 설정에서 등록된 방문목적 목록에서 선택
    - "직접입력" 선택 시 Input 필드 활성화
  - 방문 일자: 방문 날짜
  - 방문 시간: 방문 시간
  - 담당자: 내부 담당자 선택
- **특수 기능**:
  - D.I 준수 교육: 서명 또는 N/A 선택
  - N/A 선택 시 사유 입력 필수
  - ALCOA 원칙 확인 다이얼로그 표시
- **상태 관리**:
  - 대기: 등록 완료, 승인 대기
  - 승인: 출입 승인 완료
  - 거부: 출입 거부
  - 완료: 방문 완료

#### 3. 사전방문등록 탭
- **기능**: 방문 전 사전 예약 등록
- **입력 항목**: 방문 등록과 동일하나 방문 시간은 선택사항
- **사용 목적**: 
  - 사전에 방문 일정을 확정하여 효율적인 출입 관리
  - 방문 당일 "방문 등록" 탭에서 예약 정보를 불러와 빠르게 등록 가능

#### 4. 방문 이력 탭
- **기능**: 과거 방문 기록 조회
- **검색 기능**:
  - 날짜 범위 검색
  - 방문자명, 담당자, 소속, 방문목적 검색
  - 상태별 필터
- **Excel 다운로드**: 검색 결과를 Excel 파일로 다운로드

### 설정 기능
- **소속 관리**: 자주 사용하는 소속을 등록하여 드롭다운으로 빠르게 선택
  - 소속 추가, 수정, 삭제
- **방문목적 관리**: 자주 사용하는 방문목적을 등록
  - 방문목적 추가, 수정, 삭제
- **N/A 사유 관리**: D.I 준수 교육 N/A 선택 시 사용할 사유 등록
- **Audit Trail**: 모든 작업 이력 조회
  - 작업 시간, 작업 유형, 작업자, 변경 내용 등 상세 기록
  - Excel 다운로드 가능

### 예약 완료 처리
- 사전방문등록된 방문자가 실제 방문 시:
  1. 대시보드에서 해당 예약 선택
  2. "방문 등록 완료" 버튼 클릭
  3. 예약 정보가 자동으로 입력된 폼에서 확인 및 수정
  4. D.I 준수 교육 서명 또는 N/A 선택
  5. 등록 완료

## 시험장비 모니터링

시험장비 모니터링 기능을 통해 장비별 가동현황 및 가동률을 실시간으로 모니터링하고 분석합니다.

### 주요 기능

#### 1. 실시간 모니터링 탭
- **기능**: 현재 장비 상태를 실시간으로 확인
- **표시 내용**:
  - 가동 중인 장비 수 (장비 유형별 통계)
  - 전체 장비 가동률
  - 이번 달 누적 가동률
- **알림 기능**:
  - 장시간 미사용 장비 알림
  - 비정상 상태 장비 알림
- **데이터 수집**:
  - 자동 수집: 5초마다 자동으로 CDS 시스템에서 데이터 수집
  - 수동 수집: "수동 수집" 버튼으로 즉시 데이터 갱신

#### 2. 가동률 분석 탭
- **기능**: 장비별 가동률 통계 및 추이 분석
- **차트 종류**:
  - 장비별 가동률 막대 그래프
  - 기간별 가동률 추이 선 그래프
- **기간 선택**: 날짜 범위 선택으로 원하는 기간 분석
- **필터 기능**: 특정 장비만 선택하여 분석 가능

#### 3. 투자 효율성 탭
- **기능**: 장비별 투자 대비 효율성 분석
- **표시 내용**:
  - 장비 구매 비용 대비 가동률
  - ROI (투자 수익률) 분석
  - 비용 대비 효율성 산점도
- **비용 관리**: 
  - 장비별 구매 비용 등록 및 관리
  - CSV 파일 업로드로 일괄 등록 가능

#### 4. 사용자 분석 탭
- **기능**: 사용자별 장비 사용 현황 분석
- **표시 내용**:
  - 사용자별 가동 시간 통계
  - 사용자별 사용 장비 분포
  - 사용자별 가동률 비교

#### 5. 상세 데이터 탭
- **기능**: 수집된 모든 데이터를 테이블 형태로 상세 확인
- **표시 내용**:
  - 집계일자
  - 장비명
  - 시험자
  - 분석 방법 (acquisitionMethod)
  - 샘플명
  - 가동시간
  - 세션 시작/종료 시간
- **필터 기능**: 날짜, 장비, 사용자별 필터링
- **Excel 다운로드**: 필터링된 데이터를 Excel 파일로 다운로드

#### 6. 자동 보고서 탭
- **기능**: 기간별 가동률 보고서 자동 생성
- **보고서 내용**:
  - 기간 요약
  - 장비별 가동률
  - 주요 통계 지표
  - 추이 분석
- **보고서 다운로드**: PDF 또는 Excel 형식으로 다운로드 가능

### 분석 방법 매핑
- **기능**: CDS 시스템의 분석 방법을 사용자 정의 분류로 매핑
- **사용 방법**:
  1. "분석 방법 매핑" 버튼 클릭
  2. 분석 방법과 분류를 매핑
  3. 매핑 정보는 가동률 분석에 반영

### 데이터 수집
- **CDS 시스템 연동**: OpenLab Sample Scheduler API v2를 통한 실시간 데이터 수집
- **수집 주기**: 5초마다 자동 수집
- **수집 데이터**:
  - 장비 상태 (가동중, 대기, 점검중)
  - 세션 정보 (시작/종료 시간)
  - 사용자 정보
  - 분석 방법
  - 샘플 정보

## 시험장비 가동현황

시험장비 가동현황 기능은 실시간 장비 사용 현황을 모니터링합니다.

> **참고**: 이 기능은 "시험장비 모니터링" 기능으로 통합 예정입니다. (삭제예정)

### 주요 기능
- **실시간 상태 표시**: 
  - 가동중: 현재 사용 중인 장비 수
  - 대기: 사용 가능한 장비 수
  - 점검중: 점검 중인 장비 수
- **장비 목록**: 각 장비의 현재 상태 확인
- **상태별 필터링**: 가동중/대기/점검중별로 필터링하여 확인

### 사용 방법
1. 장비 목록에서 각 장비의 현재 상태 확인
2. 상태별 필터를 사용하여 원하는 상태의 장비만 표시
3. 장비명 클릭 시 상세 정보 확인

## 공통 기능

### 데이터 갱신
- 모든 페이지는 5초마다 자동으로 데이터를 갱신합니다

### 필터 및 검색
- 날짜 범위 선택
- 키워드 검색
- 드롭다운 필터
- 여러 조건 조합 검색

### 반응형 디자인
- 데스크톱, 태블릿, 모바일 환경 모두 지원
- 화면 크기에 따라 자동으로 레이아웃 조정
